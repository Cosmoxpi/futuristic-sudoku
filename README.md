## рҹҡҖ Getting Started

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/Cosmoxpi/neon-sudoku-app.git
```

### 2. Open the Project Folder

```bash
cd neon-sudoku-app
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

Now open the local URL shown in your terminal, usually:

```text
http://localhost:8080
```

## рҹ§© Gameplay Overview

1. Open the app and click **Start Game**.
2. Select a difficulty level.
3. Choose an empty cell on the Sudoku board.
4. Enter a number using the keypad.
5. Use hints, undo, redo, or erase whenever needed.
6. Complete the grid and try to beat your best local score.
